# Portfolio

- ポートフォリオのワークアウトアプリです

## スタック

- React.js 
- Firebase


## 環境構築

- このリポジトリを`git clone`
- `yarn install`
- `yarn start`でローカル環境が立ち上がることを確認

## 開発ルール

### コンポーネント管理

atomic designを採用しています
`src/components`配下に以下ディレクトリがあるので、適切箇所にファイル実装してください。

- atoms
- molecules
- organisms
- templates
